Why Sway-Draw?

⚡

Blazing Performance

Partial damage tracking ensures only modified areas are redrawn, even on 4K displays. Minimal CPU usage, maximum responsiveness.

🛡️

Native Wayland

Built with wlr-layer-shell and smithay-client-toolkit for a seamless integration with Sway, Hyprland, and other compositors.

🖌️

Smooth as Ink

Multi-pass Laplacian smoothing combined with Quadratic Bézier curves for professional-looking freehand annotations.

🪶

Ultra-Lightweight

Written in Rust with software rendering via tiny-skia. No heavy UI toolkits or GPU dependencies required.

Control at Your Fingertips

Ctrl + 1 Freehand Tool
Ctrl + 2 Rectangle Tool
Ctrl + 3 Line & Arrow
Ctrl + 4 Toggle Smoothing
Ctrl + Z Undo last stroke
Ctrl + D Clear Screen
Esc Exit Application

Quick Installation

bash
git clone https://github.com/vedanshbodkhe21/sway-draw.git
cd sway-draw
cargo run --release

Requires a Wayland compositor that supports wlr-layer-shell.
